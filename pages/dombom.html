html DOM

html DOM er en forkortelse for Document Object Model. DOM bliver oprettet når en webside er indlæst. DOM udseeendet er som et træ med parents, leafs og en root.
html DOM deffinere det følgende:

  • HTML elementer som objekter
  • Proterties for alle HTML elementerne
  • Metoder for alle HTML elementerne
  • Events for HTML elementerne

For JavaScript er html DOM et API, hvilket vil sige det er et programmerings interface.

Det betyder JavaScript kan:

  • tilføje, ændre, slette HTML elementer
  • tilføje, ændre, slette HTML attributter
  • tilføje, ændre, slette CSS styles
  • reagere på HTML events
  • tilføje, ændre, slette HTML events

JS browser BOM

BOM er en forkortelse for Browser Object Model. DOM gør det muligt for Javascript at kommunikere med browseren.
