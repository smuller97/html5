Bootstrap

Bootstrap er et mobilefirst CSS framework. Det vil sige at bootstrap tillader nemmere devlopment til alle skærmstørrelser. Dette opnåes via brug af det 12 colloner brede grid. Griddet kan herefter modificeres til de 5 skærmstørrelsesvider bootstrap baseres på.


Bootstrap indeholder også en basal css, som gør din side pænnere


Kode eksempler

.Container

Container er en class lavet af bootsrap, som centrerer dit indhold, og som ændrer størrelse altefter skærmstørrelse.

.col-sm-3

Koden her bestemer hvor meget af breden på 12, som denne del af koden skal indeholde på skræmbilleder større end ekstra small skærme


Her benyttes css grid, som bootstrap udbygger med skærmstørrelser. Når man er i container elementet, så har man adgang til columns og rows. Disse giver en fleksibel, og relativ nem måde at konstruere en mobilvenlig side

.mb-3

Dette er margin bottom, virker som vi kender det ved at tilføje margin til bunden af elementet

.bg-dark

Ændrer baggrunden af elementet til farven dark
