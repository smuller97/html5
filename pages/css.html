Forklaring

CSS er en forkortelse for Cascading Style Sheets. CSS formål er at bestemme hvordan hjemmesiden skal se ud. Også hvordan den ser ud på forskellige enheder. Det er nemt at finde koden, let at ændre og kan hurtigt ændre udseendet på din hjemmeside

Der findes 3 typer af css

  • Inline
  • Eksternal
  • Internal

Inline

Inline CSS er hvor CSS er lige ved det element der skal ændres


Eksternal

Eksternal CSS er hvor CSS er gemt i en fil for sig selv. Det refereres med et stylesheet i <head> tagget. En god ide er at lave en kommentar i koden, så man bedre får et overblik af hvor referencen befinder sig. Det kan eksempelvis være <!--Stylesheet-->


Internal

Internal CSS erhvor CSS er i toppen af HTML siden i dens egen tag <style> </style>
