HTML5

Beskrivelse af HTML5:

  • Det er den femte hovedstandard indenfor HTML.
  • Udvikling af sproget startede i 2007
  • Endelig HTML5 standard officiel 28.oktober 2014 af W3C (World wide web consortium)
  • Tim Berners-Lee - CEO i W3C og opfidner af WWW (World Wide Web)
  • HTML 5 er designet til af afløse XHTML og en HTML version, der blev udgivet i 2000.
  • HTML5 står for Hypertext Markup Language revision 5
  • Det er et sprog for struktur og præsentation af det store internets indhold.

Kilde: Techopedia HTML


CSS

Beskrivelse af CSS3:

  • Det er den tredje hovedstandard indenfor CSS.
  • Udvikling af CSS3 startede i 1998, og var endeligt udgivet i 2009
  • Håkon Wium Lie - foreslog orginalt css, og Bert Bos er bliver anset som co creator
  • Ideen i css er at skille layout og content fra hinanden.
  • CSS3 har ændret strukturen, så den består af moduler, som sidenhen har gjort at modulerne udvikler sig uafhængigt af hinanden.
  • CSS3 er backwards compatible
